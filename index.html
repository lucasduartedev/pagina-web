Desenvolvimento WEB

{ HTML & CSS }

Página projetada com o objetivo de aplicar conhecimentos sobre desenvolvimento WEB obtidos na Formação: A partir do zero: HTML e CSS para projetos web do Alura.

Para construír uma página na WEB usamos o HTML5, uma linguagem de marcação, possibilitando estruturar páginas WEB usando suas tag's semânticas.

Com o CSS3 Cascading Style Sheets (Folhas de Estilo em Cascata), linguagem de estilos, podemos 'dar vida' com a estilização das páginas melhorando sua apresentação aplicando cores, fontes, posicionamento de elementos e etc.

Nessa página não foi aplicado nenhum código Javascript, foram aplicadas apenas técnicas de desenvolvimento web com HTML 5 e CSS 3.
